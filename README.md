# SWGOH GAC Helper

A lightweight mobile-first GAC assistant for Star Wars Galaxy of Heroes.

## Current Features

- Google Sheets backend
- PWA Home Screen app
- 5v5 / 3v3 support
- Counter lookup
- Live updates from Google Sheets
